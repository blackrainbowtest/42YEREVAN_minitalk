Detailed Description

Signal handler to acknowledge receipt of a signal from the server.

Entry point for the client program.

Sends a character to the server bit by bit.

Sends a single bit to the server and waits for confirmation.

Waits for a specific ACK value from the server.

Signal handler for acknowledging signal receipt.

Parameters
